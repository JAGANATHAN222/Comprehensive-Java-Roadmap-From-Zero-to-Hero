# Java Mastery: Zero to Hero 🚀

Java கற்கும் பயணத்தை (Learning Journey) மற்றும் பயிற்சியை (Practice) முறையாக சேமித்து வைப்பதற்கான ஒரு முழுமையான Repository. இதில் அடிப்படை முதல் மேம்பட்ட (Advanced) கருத்துக்கள் வரை அனைத்தும் விரிவாக உள்ளது.

---

### 📌 Repository Overview
இந்த Repo-வில் Java-வின் அனைத்து முக்கிய பகுதிகளும் பாயிண்ட் வாரியாக பிரிக்கப்பட்டுள்ளன:

* **Level 1: Basics (அடிப்படைகள்)**
    * Syntax, Variables, and Data Types
    * Operators and Expressions
    * Control Flow (If-Else, Switch, Loops)

* **Level 2: OOPs Concepts (பொருள்சார் நிரலாக்கம்)**
    * Classes and Objects
    * Inheritance, Polymorphism, Abstraction, and Encapsulation
    * Constructors and 'this' keyword

* **Level 3: Core Java (முக்கிய பகுதிகள்)**
    * Exception Handling (Try-Catch, Finally)
    * String Handling and Wrapper Classes
    * File Input/Output (I/O)

* **Level 4: Collections Framework**
    * List (ArrayList, LinkedList)
    * Set (HashSet, TreeSet)
    * Map (HashMap, TreeMap)

* **Level 5: Advanced Java (மேம்பட்டவை)**
    * Multithreading and Concurrency
    * Lambda Expressions and Functional Interfaces
    * Stream API and Optionals

* **Level 6: Mini Projects**
    * Console-based Applications
    * Logic Building Exercises

### 🛠️ How to Run
இந்த கோப்புகளை உங்கள் கணினியில் இயக்க:
1. இந்த ரெப்போவை குளோன் செய்யவும்: `git clone <your-repo-link>`
2. குறிப்பிட்ட ஃபோல்டருக்கு செல்லவும்.
3. கம்பைல் செய்ய: `javac FileName.java`
4. இயக்க: `java FileName`
